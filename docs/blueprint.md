# **App Name**: WeldSense AI Dashboard

## Core Features:

- Real-time Production & Robot Dashboard: Displays key performance indicators (KPIs) for welding production status, robot operational status, and overall system efficiency through dynamic charts and tables.
- Interactive Production Calendar: Provides a calendar interface for viewing, creating, and updating daily or weekly production schedules, tasks, and maintenance events. Includes filtering capabilities by order, personnel, and specific robots.
- AI-Powered Production Anomaly Detection: An AI tool that analyzes historical and real-time production data to identify potential anomalies, predict future bottlenecks, or detect deviations from expected performance, providing insights or alerts.
- Robot Asset & Status Monitoring: Manages information and tracks the real-time operational status (operating, standby, unavailable) of individual welding robots, including basic specifications and historical data.
- Data Storage and Retrieval: Implements a backend database to persist and retrieve all operational data, including production schedules, robot statistics, KPIs, and AI-generated insights.

## Style Guidelines:

- The application adopts a dark color scheme to emphasize its high-tech, industrial monitoring purpose, suitable for prolonged viewing. The primary color is a deep, professional blue (#3399CC) which conveys reliability and efficiency.
- The background color is a very dark, subtly bluish-gray (#16191B), providing a clear contrast for textual and graphical data without being harsh on the eyes.
- An accent color of bright cyan (#78DBDB) is used to highlight interactive elements, active selections, and key alerts, providing a modern and clear contrast.
- Body and headline font: 'Inter' (sans-serif) for its neutral, modern aesthetic and excellent legibility, making it suitable for displaying complex data and system information.
- Use a set of minimal, modern line icons for navigation and functional elements, maintaining a clean and professional appearance that complements the high-tech industrial theme.
- A split layout design, featuring a fixed-width left navigation sidebar and a scrollable main content area that adapts to display dashboards, calendars, and detailed information. A right-hand panel is utilized for summary statistics.
- Incorporate subtle, smooth transitions for loading data, navigating between sections, and applying filters, enhancing user experience without distracting from critical information.
